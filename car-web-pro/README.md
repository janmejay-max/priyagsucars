# Car Web Pro

This is a static car website built with HTML, CSS, and JavaScript.

## Publish to the web

### Option 1: Deploy to Netlify (recommended)
1. Go to https://www.netlify.com/
2. Sign up for a free account.
3. Choose "Add new site" → "Deploy manually".
4. Drag and drop the `car web pro` project folder into the upload area.
5. Netlify will publish the site and give you a public link.

### Option 2: Deploy to GitHub Pages
1. Create a GitHub account at https://github.com/ if you do not already have one.
2. Create a new repository.
3. Upload all project files: `index.html`, `styles.css`, `script.js`, `assets/`.
4. In the repository settings, enable GitHub Pages and set the source to the `main` branch.
5. GitHub Pages will publish your site at `https://<username>.github.io/<repo-name>/`.

### Notes
- `localhost:8000` is only available on your own PC.
- To make the site public, it must be deployed to a hosting service.
- To appear on Google, first publish it publicly, then submit the site URL to Google Search Console.
